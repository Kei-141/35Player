# 35Player
## 開発経緯
ホロライブ所属Vtuberの歌枠の曲をプレイリストにして聞けないかな？という発想から開発に至りました。

## 機能
- 配信アーカイブ毎の曲一覧の表示
- プレイリストの作成
- 再生・ループ再生・曲送り・曲戻し

## 機能追加予定（多分やる）
- スマホ用UIの作成
- UIのリッチ化
- シャッフル再生・ランダム再生機能
- 再生開始・終了時間の調整機能
- 検索機能（アーティスト検索・曲検索など）

## その他の予定
- 曲データ作成用の支援ツールを開発する
